# Donthu et al. 2021 Notes

Derived from the user-provided local PDF `Donthu et al. - 2021 - How to conduct a bibliometric analysis An overview and guidelines.pdf`.

## What this source governs

This is the core procedural source for deciding when bibliometric analysis is appropriate, which technique family to use, and how to execute a defensible four-step bibliometric workflow.

## Core distinctions extracted from the local source

- bibliometric analysis is most useful for broad fields with large bibliographic corpora
- bibliometric analysis should not be forced onto small corpora that are better suited to systematic review or meta-analysis
- bibliometric analysis is complementary to meta-analysis and systematic literature review rather than a universal replacement
- a strong bibliometric study should usually combine performance analysis with science mapping rather than relying on productivity counts alone

## Technique structure highlighted in the source

- `performance analysis` explains the contributions of research constituents such as authors, journals, institutions, and countries
- `science mapping` explains the relationships between research constituents
- science-mapping techniques include citation analysis, co-citation analysis, bibliographic coupling, co-word analysis, and co-authorship analysis

## Four-step workflow extracted from the source

1. define the aims and scope of the bibliometric study
2. choose the bibliometric techniques that fit those aims
3. collect the data required by the selected techniques
4. run the analysis and report the findings together with substantive interpretation

## Methodological expectations to derive from this source

- define aim and scope before choosing data
- choose techniques first, then make sure the dataset supports those techniques
- use bibliometric analysis only when the field size justifies it
- prefer one appropriate database when possible to reduce harmonization errors
- clean duplicates and erroneous records before analysis
- interpret clusters and networks analytically rather than only descriptively

## Technique-selection guidance from the source

- use `co-citation analysis` to reveal foundational or past intellectual structure
- use `bibliographic coupling` to reveal present alignments and active fronts
- use `co-word analysis` to reveal themes, topic structure, and possible future directions
- use `performance analysis` by default as the field profile layer

## Scope caution

- if the likely corpus is only tens or low hundreds of papers, bibliometric analysis may be methodologically disproportionate
- do not claim strong qualitative conclusions from quantitative maps without careful interpretation
- do not make long-range forecasts from bibliometric outputs
