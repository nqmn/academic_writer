# Engineering Mode

## Priorities

- System performance
- Reproducibility
- Quantitative precision
- Implementation clarity

## Preferred Language

- "performance improved by ..."
- "the system was evaluated using ..."
- "the proposed method reduces ..."
- "reproducibility was supported through ..."

## Warnings

- Avoid vague claims of optimisation without metrics.
- Report constraints, failure cases, and evaluation conditions explicitly.
