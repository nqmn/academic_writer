# Passas 2024 Notes

Derived from the user-provided local PDF `encyclopedia-04-00065-v4.pdf`.

## What this source adds

This source contributes a compact seven-step operational checklist for bibliometric analysis and a practical tool-oriented view of the workflow.

## Seven-step structure extracted from the local source

1. define research objectives
2. perform literature search and data collection
3. clean and preprocess the data
4. select the bibliometric techniques
5. run the analysis
6. visualize the results
7. interpret and report the findings

## Practical tool suggestions visible in the source

- `EndNote`, `Zotero`, and `Mendeley` for reference collection and organization
- `R`, `Python`, `Excel`, and `LibreOffice` for cleaning and preprocessing
- `VOSviewer` and `CiteSpace` for technique execution and mapping
- `Bibliometrix` for advanced structural analysis and visualization
- `MS Word` and `LaTeX` for reporting

## Agent behavior to derive from this source

- keep the workflow explicit and sequential when the user asks for a plan
- separate cleaning, analysis, visualization, and interpretation rather than collapsing them into one vague step
- state the expected output of each stage so the user can see how the bibliometric design progresses
- use software recommendations pragmatically rather than treating any one tool as mandatory

## Scope caution

- this source is a concise operational guide and should be used together with Donthu et al. for methodological rationale and with Ozturk et al. for stronger research-design transparency
