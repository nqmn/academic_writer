# Interdisciplinary Mode

## Priorities

- Terminological clarity across fields
- Explicit justification of mixed frameworks
- Balanced treatment of differing evidential norms
- Careful translation between disciplinary assumptions

## Preferred Language

- "draws on insights from ..."
- "within this study, the concept is used in a more limited sense ..."
- "the framework integrates ..."
- "these findings speak to both ... and ..."

## Warnings

- Avoid importing terms from one field without definition.
- State which disciplinary standards guide evaluation in each section.
